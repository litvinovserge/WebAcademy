### HomeWork_10
- **HomeTask_1** <br />
  Переписать информацию с одного файла в другой.
- **HomeTask_2** <br />
  Скопировать файл в выбранную директорию.
- **HomeTask_3** <br />
  Определите размер файла имя которого вводил пользователь. Выведите его на экран. Выведите его в байтах, мегабайтах, гигабайтах.
- **HomeTask_4** <br />
  Написать программу запрашивающую у пользователя его имя при первом запуске. При последующих запуска программа должна обращаться по имени. Хранить имя последнего юзера в файле. 
- **HomeTask_5** <br />
  Написать программу способную регистрировать пользователей, добавить функциональность регистрации, входа и выхода. Каждый последующий запуск программы должен восстанавливать предыдущее состояние. Пользователей может быть несколько, отлавливать ситуации когда пароль введен неверно или когда нет такого юзера. Для сохранения состояния можно использовать модуль pickle или json.
